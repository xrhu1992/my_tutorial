# Markdown语法
## 基本语法
  ![基本语法](pic/markdown/基本语法.png)
* 修改图片显示的大小
  - 修改宽高 `![图片描述](https://example.com/image.png =300x200)`
  - 只修改宽，高自动等比例调整 `![图片描述](https://example.com/image.png =300x)`

## 兼容HTML语法
* 文本居中 
  `<center>text</text>`
  <center>text</text>
* 换行 
  `<br>`
* 插入图片并缩放
  `<img src="image.png" width = "300">`

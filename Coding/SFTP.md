# SFTP 常用命令
* SFTP（SSH File Transfer Protocol）是一种安全的文件传输协议，它通过 SSH（Secure Shell）加密通道传输文件。与传统的 FTP 不同，SFTP 提供了更高的安全性，所有传输的数据都会被加密。

## 文件操作

| 命令 | 说明 |
|------|------|
| `get <远程文件> [本地路径]` | 下载文件 |
| `put <本地文件> [远程路径]` | 上传文件 |
| `mget <文件1> <文件2> ...` | 批量下载文件 |
| `mput <文件1> <文件2> ...` | 批量上传文件 |
| `rm <文件>` | 删除远程文件 |
| `rename <旧名> <新名>` | 重命名远程文件 |

## 目录操作

| 命令 | 说明 |
|------|------|
| `ls` | 列出远程目录内容 |
| `lls` | 列出本地目录内容 |
| `cd <目录>` | 切换远程目录 |
| `lcd <目录>` | 切换本地目录 |
| `mkdir <目录名>` | 创建远程目录 |
| `lmkdir <目录名>` | 创建本地目录 |
| `rmdir <目录名>` | 删除远程目录 |
| `pwd` | 显示远程当前目录 |
| `lpwd` | 显示本地当前目录 |

## 其他常用

| 命令 | 说明 |
|------|------|
| `help` 或 `?` | 显示帮助 |
| `exit` 或 `quit` | 退出 SFTP |
| `progress` | 切换进度条显示 |
| `! <命令>` | 执行本地 shell 命令 |

## 连接示例

```bash
# 连接远程服务器
sftp user@hostname

# 指定端口连接
sftp -P 2222 user@hostname
```

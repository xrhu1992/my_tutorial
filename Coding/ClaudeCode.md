# Claude Code

## CLAUDE.md 配置
CLAUDE.md支持三个级别的配置，按**优先级从低到高**，Claude启动时会按顺序读取这三个文件，后面的覆盖前面的。
1. **全局配置**：~/.claude/CLAUDE.md。适用于所有项目的通用规范,如代码风格偏好、常用技术栈。
2. **用户配置**：~/CLAUDE.md(用户根目录)。用户级个人偏好。
3. **项目配置**：./CLAUDE.md(项目根目录)。项目特定的规范和上下文，优先级最高，会覆盖全局配置。


## 快捷指令
* `#` 向CLAUDE.md中添加规则
  ```
  # 使用Python作为编程语言
  ```
